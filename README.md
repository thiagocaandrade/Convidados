# APP Convidados - Kotlin

App convidados que adiciona, edita e deleta pessoas em Kotlin, app desenvolvido no curso da DevMasterTeam.

-  Arquitetura MVVM 
 - SQLiteDatabase 
 - RecyclerView 
 - Navigation Drawer
 - Fragments
